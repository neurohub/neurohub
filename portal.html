NeuroHub Portal

To request access to the NeuroHub beta, follow the “Open Portal” button above. Once on the NeuroHub Portal page, click “Request an Account” in the bottom-right to request beta access. The NeuroHub team will review your request as soon as possible.

Current features include:

  • Login (via ORCID ID or Registration)
  • Project Creation
  • Project Collaboration Invitations
  • Creator Defined Project Licences
  • Project Team Managements (Add Editors)
  • Storage Space Customization
  • User File Management (Add New Files or Connect to Collections)
  • Search Function
  • CBRAIN Links (Ability to Explore Both simultaneously)
  • User-to-User Interactions

Contact support@neurohub.ca for any issues
